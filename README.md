# LAB 01 - Mapeamento de Rede Local

# Objetivo:
Realizar o mapeamento completo de uma rede local utilizando ferramentas nativas do Windows 11, aplicando conceitos de redes TCP/IP e diagnostico de infraestrutura.

# Ambiente
- Sistema Operacional: Windows 11 Home
- Conexão: Wi-Fi Mobile Hotspot
- Rede: 10.38.68.0/24

# Ferramentas Utilizadas
- ipconfig
- arp
- ping
- Prompt de Comando

# Atividades Realizadas
- Identificação de parâmetros de rede
- Descoberta passiva via ARP
- Varredura ativa de hosts
- Identificação de Gateway
- Análise de MAC Address
- Estudo de subnetting

# Resultados
Foram identificados os dispositivos ativos na rede e confirmado o funcionamento do gateway utilizando técnicas de diagnóstico de conectividade

# Evidências
Documentação disponível na pasta '/docs'.

# Conhecimentos Aplicados
- Endereçamento IP
- Máscara de Sub-rede
- Gateway
- ARP
- Descoberta ativa e passiva
- Mac Address Randomization
